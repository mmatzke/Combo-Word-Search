# Combo-Word-Search
Finds the best spot in a PDF where the search words are closest to each other.
## Prerequisites
Install `PyMuPDF` from https://pymupdf.readthedocs.io.
```
pip install PyMuPDF
```
## Example
By searching for ["HOME", "just", "Memories"] the algorithm marks this area in 
the Sample-PDF
